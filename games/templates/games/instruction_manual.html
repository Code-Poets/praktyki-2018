Instruction manual

This application was designed to provide support during your regular Munchkin game. The application provides an interface which displays current stats of all players in the game, such as level, bonus points, gender, class and race. The host of the game should sign in and create it's panel on the website. After the panel is created, an access code for it will be generated. The host should make this code available to other players. The players are then expected to use their mobile devices to create their nicknames and access the game with given code.

And that's it! All you have to do for now on is remember to update your stats whenever they change during your game. Once any player achieves the winning level, a blockade will be set for other players and appropriate notification will be displayed on main game panel. The host can then decide to finish the game or alter it's rules, so the fun could go on. :)


Here's some additional information on the most important panels:


Homepage

From here you can got to game access view to provide your nickname and code for the game you wish to join. Logged users are additionally able to go to game hosting view to create and host a game.


Host game

Create new game by providing necessary information. You will be able to edit later if you wish. If you are already hosting any ongoing games, corresponding links will be displayed at the bottom of the page.


Game panel

An interface representing your game. It shows a table with all of the active players and their current stats. The page updates automatically. On top-right corner game's winning level is displayed next to the "Edit" button. On top-left corner a 4-character access code is displayed next to homepage redirect button. After any of the players achieves a winning level, an appropriate message will be displayed and you will be asked to either finish the game or alter it's rules in the settings view. You can access the settings at any time by choosing the "Edit" button. From there, you can change game's settings, kick players out or finish the game prematurely.

WARNING: Once the game is finished, nobody will be able to access it again.


Join game

Type in your desired nickname and game's access code. If you're already participating in any ongoing games, corresponding links will be displayed at the bottom of the page.

WARNING: Unregistered players can join only one active game at the time. If you access a new game, you will loose access to the previous one. Nicknames have to be unique in the game. You cannot join the game which is already holding your avatar (you have already joined before).


Gamer panel

Change the stats of your character in the displayed form and then hit update to save your current stats to display them on the game panel. From here you can also choose to withdraw from game. You will be able to join the game again from the join view, but all of your progress will be permanently erased.
